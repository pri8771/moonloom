# Public Beta PRD — Moonloom: Idle Dream Factory (moonloom)

**Version:** 1.0.0 | **Date:** 2026-06-27 | **Status:** Draft

---

## Overview

The Public Beta opens Moonloom to 5,000 testers via TestFlight, incorporates private beta learnings, and finalizes the game for App Store submission.

**Duration:** 4 weeks (December 1 – December 31, 2026)
**Platform:** TestFlight (iOS)
**Tester Count:** 5,000
**Build:** v0.9.x (pre-release candidate)

---

## Goals

1. Validate private beta bug fixes resolved all P0/P1 issues
2. Test at scale (5K users vs 500)
3. Validate final production balance after private beta adjustments
4. Test complete cosmetics shop and IAP flows with final art
5. Gather press preview data and screenshots
6. Get final Day-1/7/30 retention read before launch

---

## Changes from Private Beta

| Improvement | Details |
|-------------|---------|

## Success Criteria

| Metric | Target |
|--------|--------|
| Day-1 Retention | ≥ 50% |
| Day-7 Retention | ≥ 25% |
| Day-30 Retention | ≥ 12% |
| Avg. session length | ≥ 4.5 minutes |
| NPS Score | ≥ 60 |
| Crash-free rate | ≥ 99.5% |
| P0/P1 bugs | 0 |
| IAP conversion (sandbox) | ≥ 5% |
| App Store rating readiness | Preview avg ≥ 4.5 |

## Press Preview Program

- 20 gaming journalists receive press codes (Week 3 of Public Beta)
- Embargo date: January 14, 2027 (1 day before launch)
- Assets provided: screenshots, trailer, fact sheet, founder quotes
- Target publications: TouchArcade, PocketGamer, 148Apps, AppAdvice, MacRumors

---

## App Store Submission Checklist

- [ ] App icon (all sizes, no alpha)
- [ ] 6 App Store screenshots (iPhone 6.7", 6.1", 5.5")
- [ ] App Preview video (30 seconds)
- [ ] App description (US English, 4000 chars max)
- [ ] Keywords (100 chars max)
- [ ] Privacy policy URL
- [ ] Age rating questionnaire completed (4+)
- [ ] In-app purchases configured in App Store Connect
- [ ] Game Center leaderboards configured (if any)
- [ ] Subscription details + cancellation policy
- [ ] Review notes submitted

---

## Launch Readiness Gate

To exit Public Beta and proceed to App Store submission:
1. All P0 and P1 bugs resolved
2. All success criteria met
3. Press preview complete
4. App Store submission assets ready
5. Marketing launch plan confirmed
