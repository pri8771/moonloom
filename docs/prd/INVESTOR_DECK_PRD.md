## Slide 2: The Problem

**The idle game market is $4.2B — and it looks terrible.**

- Idle games are the #3 most played mobile game genre
- Top titles (AdVenture Capitalist, Cookie Clicker) use 10-year-old visual design
- Players report wanting idle games that "feel like real games"
- Average idle game retention crashes at Day-30: only 8%
- No idle game has successfully combined AAA aesthetics with prestige depth

**The Gap:** Beautiful idle games don't exist. Players are waiting.

## Slide 3: Our Solution

**Moonloom** is the first idle game that feels like a premium title.

### What We Built
- 12-tier idle factory with a moon restoration narrative
- Infinite depth via New Moon Reset (prestige) mechanic
- Offline-first: works 100% without internet
- AAA-quality art and animations at indie price point
- Fair monetization: cosmetics only, no pay-to-win

### Why It Works
- Idle mechanics are scientifically proven addictive (variable reward schedule)
- Beautiful aesthetics drive organic sharing (TikTok/Reddit)
- Narrative investment drives LTV beyond typical idle games

## Slide 4: Product Overview

### Core Loop
Sleeping Towns → Whispers → Dreamthread → Dreams → Moths → Moonlight → Moon Restoration → New Moon Reset → Permanent upgrades → Start again, stronger

### 12 Production Tiers
Whisper Nets → Lullaby Wells → Dreamthread Spindles → Memory Looms → Nightmare Filters → Star Dye Vats → Moth Courier Nests → Cloud Packaging → Dream Atlas → Comet Dock → Lucid Observatory → Moonheart Engine

### Prestige System
"New Moon Reset" — players reset for Lucid Shards, unlock permanent upgrades, restart faster. Creates near-infinite replayability.

### Offline Earnings
Factory produces while app is closed. Return after 8 hours to a pile of resources. "Your moths have been busy."
