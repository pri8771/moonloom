## Executive Summary

Moonloom: Idle Dream Factory is a premium-aesthetic idle/incremental mobile game targeting the $8.4B mobile gaming market. The game combines the proven idle game engagement loop (idle mechanics, prestige systems) with a stunning visual design and emotional narrative that differentiates it from competitors like AdVenture Capitalist and Cookie Clicker.

**Business Model:** Free-to-play with optional in-app purchases (cosmetics + convenience)
**Platform:** iOS (Android port TBD post-Year 1)
**Target Launch:** Q1 2027
**Revenue Target Year 1:** $480,000

## Product Description

Moonloom is an idle/incremental game with 12+ production tiers, a prestige system (New Moon Reset), and a world restoration narrative. Players build a dream factory and restore the moon.

### Core Differentiators
1. **Visual Quality:** AAA-level art for an indie game
2. **Emotional Narrative:** World restoration gives meaning to idle mechanics
3. **Fair Monetization:** No pay-to-win, cosmetics only
4. **Prestige Depth:** New Moon Reset provides infinite replayability
5. **Offline First:** Zero network dependency — works anywhere

## Revenue Model

### Primary Revenue Streams

#### Tier 1: Cosmetics (50% of revenue)
- Factory Themes: $2.99 each (visual reskin of entire factory)
- Moth Skins: $1.99 each (visual variants for courier moths)
- Building Skins: $1.99-$3.99 (individual building visual variants)

#### Tier 2: Subscription (25% of revenue)
- Moonloom Pass: $4.99/month
  - 2x offline earnings multiplier
  - Exclusive monthly cosmetic
  - No advantage in core mechanics

#### Tier 3: Premium Currency (15% of revenue)
- Stardust bundles: $0.99 / $2.99 / $7.99
- Used to unlock cosmetics without waiting for specific events

#### Tier 4: Convenience (10% of revenue)
- Offline Expansion: $3.99 (offline cap: 2h → 12h, one-time)
